# sdr-install
sdr linux installation

## Utilisation

1. Donnez les permissions d'exécution au script :

`sudo chmod +x install.sh`

2. Exécutez le script d'installation :

`sudo ./install.sh`

## Avertissement

Ce script est destiné à un usage éducatif et de développement uniquement.
